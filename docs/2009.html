Empirical Cloud (2009)

This is a graph of owl:sameAs links between hosts in the 2009 Billion Triple Challenge data. It is probably best viewed with a webkit browser like Chrome or Safari.

In order to make the graph somewhat viewable it was limited to hosts that shared 10 or more links. You should be able to zoom/pan on portions of the graph, and hover over a node to see the hostname. Node colors reflect org, com, gov, edu, net, other.

If your screen goes crazy cross your fingers and try to reload the page a couple times. More about how this graph was generated can be found on Github. The underlying data has been made available using VoID rdf/xml and Turtle documents.

Comments, Question, Quibbles welcome on Github. The 2010 view is also available.
